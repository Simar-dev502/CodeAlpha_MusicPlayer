# CodeAlpha_MusicPlayer

# 🎵 Melodify - Advanced Music Player

A modern and responsive web-based music player built using **HTML, CSS, and JavaScript**. Melodify provides an intuitive music listening experience with playlist management, album categorization, playback controls, shuffle mode, repeat modes, volume control, and a sleek dark-themed user interface.

## 🚀 Live Demo

🔗 melodifymusicweb.netlify.app
🔗 https://code-alpha-music-player-ruddy.vercel.app/

---

## 📌 Features

### 🎶 Music Playback

* Play / Pause songs
* Next & Previous track navigation
* Auto-play next track
* Interactive progress bar with seeking
* Real-time duration tracking

### 📂 Album Management

* Multiple music categories/albums
* Dynamic album switching
* Album-specific playlists
* Active song highlighting

### 🔀 Playback Controls

* Shuffle Mode
* Repeat One
* Repeat All
* Smart track navigation

### 🔊 Volume Controls

* Increase volume
* Decrease volume
* Live volume percentage display

### 🎨 User Interface

* Modern dark theme
* Responsive design
* Smooth animations
* Album artwork display
* Playlist panel
* Settings & menu modal

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/yourusername/CodeAlpha_MusicPlayer.git
```

2. Open project folder

```bash
cd CodeAlpha_MusicPlayer
```

3. Run using Live Server

OR simply open:

```bash
index.html
```

in your browser.
